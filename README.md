## TestProcessor

This is a simple text processor that reads a text file and outputs some relevant information about the contents of the file. This program also allows the user to alter the contents of the file.
--- 
# Features
- Find unique: displays all words in the file that are unique (single occurrence).
- Find number of occurrences: displays the number of times each word appeared in the file.
- Find and replace: locates a user-given word and replaces all occurrences of that word with another user-given word.
